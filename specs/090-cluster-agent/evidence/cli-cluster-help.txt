$ shep cluster

Usage: shep cluster [options] [command]

Manage clusters

Options:
  -h, --help                 display help for command

Commands:
  new [options] <name>       Create a new cluster
  ls                         List clusters
  show <id>                  Display details of a cluster
  del [options] <id>         Delete a cluster
  link <cluster> <entity>    Link a repository or application to a cluster
  unlink <cluster> <entity>  Unlink a repository or application from a cluster
  status <id>                Show live cluster status
  help [command]             display help for command
