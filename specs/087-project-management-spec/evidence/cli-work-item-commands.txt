=== shep item --help ===
Usage: shep item [options] [command]

Manage work items

Options:
  -h, --help                          display help for command

Commands:
  ls <project>                        List work items in a project
  new [options] <project>             Create a new work item
  relate [options] <source> <target>  Create a relation between two work items
  export [options] <project>          Export work items as CSV
  help [command]                      display help for command

=== shep item new --help ===
Usage: shep item new [options] <project>

Create a new work item

Arguments:
  project                          Project slug or identifier prefix

Options:
  -t, --title <title>              Work item title
  -p, --priority <priority>        Priority (none, low, medium, high, urgent)
  -d, --description <description>  Work item description
  --parent <identifier>            Parent work item identifier (e.g. PROJ-42)
  -h, --help                       display help for command

=== shep item ls --help ===
Usage: shep item ls [options] <project>

List work items in a project

Arguments:
  project     Project slug or identifier prefix

Options:
  -h, --help  display help for command

=== shep item export --help ===
Usage: shep item export [options] <project>

Export work items as CSV

Arguments:
  project               Project slug or identifier prefix

Options:
  -c, --columns <cols>  Comma-separated columns
                        (identifier,title,state,priority,labels,estimate,dueDate,startDate,description)
                        (default:
                        "identifier,title,state,priority,estimate,dueDate")
  -o, --output <file>   Write to file instead of stdout
  -h, --help            display help for command

=== shep item relate --help ===
Usage: shep item relate [options] <source> <target>

Create a relation between two work items

Arguments:
  source             Source work item identifier (e.g. PROJ-1)
  target             Target work item identifier (e.g. PROJ-2)

Options:
  -t, --type <type>  Relation type (blocking, relates-to, duplicate,
                     starts-before, finishes-before)
  -h, --help         display help for command
