=== shep intake --help ===
Usage: shep intake [options] [command]

Manage intake triage items

Options:
  -h, --help              display help for command

Commands:
  ls [options] <project>  List intake items for a project
  accept <id>             Accept an intake item and convert it to a work item
  decline [options] <id>  Decline an intake item with a reason
  help [command]          display help for command

=== shep intake ls --help ===
Usage: shep intake ls [options] <project>

List intake items for a project

Arguments:
  project                Project slug or identifier prefix

Options:
  -s, --status <status>  Filter by status (Pending, Accepted, Declined,
                         Duplicate)
  -h, --help             display help for command

  id          Intake item ID

Options:
  -h, --help  display help for command

=== shep intake decline --help ===
Usage: shep intake decline [options] <id>

Decline an intake item with a reason

Arguments:
  id                     Intake item ID

Options:
  -r, --reason <reason>  Reason for declining
  -h, --help             display help for command
