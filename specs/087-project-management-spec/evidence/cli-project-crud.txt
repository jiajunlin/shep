=== shep project --help ===
Usage: shep project [options] [command]

Manage projects

Options:
  -h, --help            display help for command

Commands:
  ls                    List all projects
  new [options]         Create a new project
  show <slug>           Show project details
  del [options] <slug>  Delete a project
  help [command]        display help for command

=== shep project new --help ===
Usage: shep project new [options]

Create a new project

Options:
  -n, --name <name>                Project name
  -p, --prefix <prefix>            Identifier prefix (1-5 uppercase letters)
  -d, --description <description>  Project description
  -h, --help                       display help for command
