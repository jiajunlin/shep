(Global search is a web UI feature accessible via Cmd+K shortcut.
 It uses the SearchWorkItems FTS5 use case. The CLI demonstrates
 the underlying ListWorkItems and search-related infrastructure.)

=== All PM commands registered in CLI ===
Usage: shep [options] [command]

Autonomous AI Native SDLC Platform - Automate the development cycle from idea to
deploy

Options:
  -v, --version               Display version number
  -h, --help                  display help for command

Commands:
  version                     Display version information
  settings                    Manage Shep global settings
  ui [options]                Start the Shep web UI
  run [options] <agent-name>  Run an AI agent workflow
  agent                       Manage and view agent runs
  feat                        Manage features through the SDLC lifecycle
  repo                        Manage tracked repositories
  session                     Manage and view agent provider CLI sessions
  ide [options] <feat-id>     Open a feature worktree in your IDE
  install [options] [tool]    Install a development tool
  tools                       Manage development tools
  project                     Manage projects
  item                        Manage work items
  cycle                       Manage sprint cycles
  intake                      Manage intake triage items
  notifications|notif         Manage notifications
  upgrade                     Upgrade Shep CLI to the latest version
  start [options]             Start the Shep web UI as a background daemon
  stop                        Stop the running Shep web UI daemon
  restart [options]           Gracefully restart the Shep web UI daemon (starts
                              it if not running)
  status [options]            Show the status of the Shep web UI daemon
