=== shep cycle --help ===
Usage: shep cycle [options] [command]

Manage sprint cycles

Options:
  -h, --help                            display help for command

Commands:
  ls <project>                          List cycles in a project
  new [options] <project>               Create a new cycle
  show <cycle-id>                       Show cycle details
  add-items <cycle-id> <item-ids...>    Add work items to a cycle
  transfer [options] <source> [target]  Transfer incomplete items from one cycle
                                        to another
  help [command]                        display help for command

=== shep cycle new --help ===
Usage: shep cycle new [options] <project>

Create a new cycle

Arguments:
  project                          Project slug or ID

Options:
  -n, --name <name>                Cycle name
  -d, --description <description>  Cycle description
  -s, --start <date>               Start date (YYYY-MM-DD)
  -e, --end <date>                 End date (YYYY-MM-DD)
  -h, --help                       display help for command

=== shep cycle ls --help ===
Usage: shep cycle ls [options] <project>

List cycles in a project

Arguments:
  project     Project slug or ID

Options:
  -h, --help  display help for command

=== shep cycle show --help ===
Usage: shep cycle show [options] <cycle-id>

  cycle-id    Cycle ID

Options:
  -h, --help  display help for command

=== shep cycle add-items --help ===
Usage: shep cycle add-items [options] <cycle-id> <item-ids...>

Add work items to a cycle

Arguments:
  cycle-id    Cycle ID
  item-ids    Work item IDs to add

Options:
  -h, --help  display help for command

=== shep cycle transfer --help ===
Usage: shep cycle transfer [options] <source> [target]

Transfer incomplete items from one cycle to another

Arguments:
  source       Source cycle ID
  target       Target cycle ID (omit to remove items from cycle)

Options:
  -f, --force  Skip confirmation prompt
  -h, --help   display help for command
