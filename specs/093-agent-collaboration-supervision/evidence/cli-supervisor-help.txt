=== shep supervisor --help ===
Usage: shep supervisor [options] [command]

Manage the delegated supervisor agent (spec 093)

Options:
  -h, --help           display help for command

Commands:
  configure [options]  Create or update a supervisor policy
  status [options]     Show the effective supervisor policy for an app or
                       feature
  enable [options]     Enable an existing supervisor policy
  disable [options]    Disable an existing supervisor policy
  approve [options]    Approve a waiting agent run as the supervisor
  reject [options]     Reject a waiting agent run as the supervisor
  help [command]       display help for command

=== shep agent questions --help ===
Usage: shep agent questions [options] [command]

Manage agent questions raised during interactive and background runs

Options:
  -h, --help                     display help for command

Commands:
  ls [options]                   List agent questions for an app or feature
  answer [options] <questionId>  Submit an answer for a pending agent question
  cancel [options] <questionId>  Cancel a pending agent question
  help [command]                 display help for command

=== shep agent message --help ===
Usage: shep agent message [options] [command]

[dev] inspect and inject inter-agent messages on the bus

Options:
  -h, --help      display help for command

Commands:
  send [options]  [dev] publish a single inter-agent message on the bus
  help [command]  display help for command
