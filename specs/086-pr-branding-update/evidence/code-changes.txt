CODE CHANGES FOR PR BRANDING UPDATE

=== MODIFIED FILE ===
packages/core/src/infrastructure/services/git/pr-branding.ts

=== BEFORE ===
export const PR_BRANDING =
  'Built with Shep 🐑 [Shep Bot](https://github.com/shep-ai/shep)';

=== AFTER ===

=== DETAILED COMPARISON ===

Old Format:
- Text: "Built with Shep 🐑 [Shep Bot]"
- Links: Only Shep Bot text links to GitHub
- Emoji: Unlinked sheep emoji
- Style: Plain text prefix

New Format:
- Text: "[🐑] Built with [Shep.bot]"
- Links: Sheep emoji links to GitHub repo, Shep.bot links to shep.bot domain
- Emoji: Linked and prominent at start
- Style: Markdown link format

=== CHANGES MADE ===
1. Moved sheep emoji (🐑) to the beginning of the branding line
2. Wrapped emoji in markdown link: [🐑](https://github.com/shep-ai/shep)
3. Updated "Shep Bot" to "Shep.bot" (better branding)
4. Changed Shep.bot to link to https://shep.bot (product domain)
5. Reordered text to: emoji + branding text
