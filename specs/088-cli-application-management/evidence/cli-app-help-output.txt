$ shep app --help

Usage: shep app [options] [command]

Manage applications

Options:
  -h, --help                   display help for command

Commands:
  ls                           List applications
  show <id>                    Display details of an application
  new [options] <description>  Create a new application
  del [options] <id>           Delete an application
  help [command]               display help for command

================================================================================

$ shep app new --help

Create a new application

Arguments:
  description      Application description

Options:
  --agent <type>   Agent type override
  --model <model>  LLM model override (e.g. claude-opus-4-6)
  --no-start       Create without starting an agent session
  -h, --help       display help for command
